d3-group

dynamic line() accessors

For each line to plot, there is a need to change the y-accessor of line() dynamically based on the group. Example: line().y((d) => yScale(d.m1)) for m1, and line().y((d) => yScale(d.m2)) for m2, etc. Use d[category] instead of using dot notation.
